Hoe is deze website gemaakt?

13 mei 2021

Template en hosting

Deze website is begonnen als het Strada template van HTML5 UP dit is een gratis te gebruiken template die bedoeld is om te gebruiken als portfolio voor een fotograaf. Maar mits wat kleine aanpassingen was er perfect een portfolio voor een IT’er van te maken. Het thema kwam met een index pagina, placeholder foto’s en css en javascript. De blogpagina’s en het aanpassen van het vergroten van foto’s naar het doorlinken naar een pagina heb ik zelf gedaan. Dit is niet zoveel werk als je een behoorlijke basiskennis hebt van html, CSS en JS.

Deze website wordt gehost via Vercel en de domeinregistratie gebeurd via een gratis domein die ik via Howest kon registreren bij Combell.

Een blogpost toevoegen

Een post is zeer gemakkelijk toe te voegen, je neemt een kopie van een bestaande blogpost, je maakt een nieuw mapje aan en zet daar je gekopieerde index in. Eens je dit gedaan hebt hoef je enkel nog Een foto en titel met link op de homepagina toe te voegen. Tot slot moet je ook nog alle tekst in je gekopieerde index pagina plaatsen en voilà, je hebt een nieuwe blogpost gecreëerd.
